# Referensi Tipe TypeScript

Semua tipe yang diekspor dari starter kit.

## Tipe Konfigurasi

Sumber: `src/auth/config/types.ts`

### `AuthConfig`

```typescript
interface AuthConfig {
  email: EmailConfig;
  database: DatabaseConfig;
  features: FeaturesConfig;
  session: SessionConfig;
  ui: UiConfig;
}
```

### `AuthConfigInput`

Tipe input yang diterima oleh `defineAuthConfig()`. Sama seperti `AuthConfig` tetapi dengan semua field opsional (default diterapkan).

### `EmailConfig`

```typescript
interface EmailConfig {
  sendVerificationEmail?: SendVerificationEmailCallback;
  sendPasswordResetEmail?: SendPasswordResetEmailCallback;
}

type SendVerificationEmailCallback = (params: {
  email: string;
  url: string;
  token: string;
}) => Promise<void>;

type SendPasswordResetEmailCallback = (params: {
  email: string;
  url: string;
  token: string;
}) => Promise<void>;
```

### `DatabaseConfig`

```typescript
interface DatabaseConfig {
  adapter: "prisma" | "drizzle" | "kysely";
  url: string;
  client?: unknown;
}
```

### `SessionConfig`

```typescript
interface SessionConfig {
  expiresIn: string;
  strategy: "jwt" | "database";
  cookieName: string;
  updateAge?: string;
  cookieCache?: boolean;
}
```

### `FeaturesConfig`

```typescript
interface FeaturesConfig {
  emailPassword: EmailPasswordConfig;
  passwordReset: PasswordResetConfig;
  oauth: OAuthConfig;
  twoFactor: TwoFactorConfig;
  roles: RolesConfig;
}
```

### `EmailPasswordConfig`

```typescript
interface EmailPasswordConfig {
  enabled: boolean;
  requireEmailVerification: boolean;
  passwordMinLength: number;
  passwordMaxLength: number;
  disableSignUp: boolean;
  autoSignIn: boolean;
}
```

### `PasswordResetConfig`

```typescript
type PasswordResetConfig = boolean | {
  expiresIn?: string;
};
```

### `OAuthConfig`

```typescript
interface OAuthConfig {
  github?: OAuthProviderConfig;
  google?: OAuthProviderConfig;
  custom?: CustomOAuthProviderConfig[];
}

interface OAuthProviderConfig {
  enabled: boolean;
  clientId: string;
  clientSecret: string;
}
```

### `TwoFactorConfig`

```typescript
interface TwoFactorConfig {
  enabled: boolean;
  methods: "totp"[];
}
```

### `RolesConfig`

```typescript
interface RolesConfig {
  enabled: boolean;
  defaultRole: string;
  roles: string[];
}
```

### `UiConfig`

```typescript
interface UiConfig {
  theme: "shadcn";
  redirectAfterLogin: string;
  redirectAfterLogout: string;
  twoFactorSettingsPath: string;
  protectedPaths: string[];
  roleRestrictions: Record<string, string[]>;
}
```

## Tipe Hasil Aksi

Sumber: `src/lib/auth/actions/types.ts`

```typescript
interface AuthSuccessResult<T = void> {
  success: true;
  message?: string;
  data?: T;
}

interface AuthErrorResult {
  success: false;
  error: {
    code: string;
    message: string;
  };
}

type AuthActionResult<T = void> = AuthSuccessResult<T> | AuthErrorResult;
```
