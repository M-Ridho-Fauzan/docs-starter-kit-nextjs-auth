# Referensi React Hooks

Semua hooks berada di sisi klien (`"use client"`) dan bekerja dengan React 19.

## Auth & Session

### `useAuth`

Mengembalikan nilai konteks auth (state session dan provider).

```tsx
import { useAuth } from "@/hooks/use-auth";

function AuthStatus() {
  const { isPending, session } = useAuth();
  if (isPending) return <div>Memuat...</div>;
  return <div>{session ? "Sudah login" : "Belum login"}</div>;
}
```

### `useSession`

Mengembalikan session saat ini dengan kemampuan refresh.

```tsx
import { useSession } from "@/hooks/use-session";

function UserEmail() {
  const { data: session, isPending } = useSession();
  if (isPending) return null;
  return <span>{session?.user?.email}</span>;
}
```

## Form Hooks

Form hooks membungkus server actions dengan `useActionState` (React 19).

### `useLoginForm`

```tsx
import { useLoginForm } from "@/hooks/use-login";

function LoginPage() {
  const [state, formAction, pending] = useLoginForm();
  return (
    <form action={formAction}>
      <input name="email" type="email" required />
      <input name="password" type="password" required />
      <button type="submit" disabled={pending}>Masuk</button>
    </form>
  );
}
```

### `useSocialLogin`

```tsx

  const { signIn } = useSocialLogin();
  return <button onClick={() => signIn("github", "/dashboard")}>GitHub</button>;
}
```

## Roles

### `useRole`

Mengembalikan string peran pengguna saat ini.

```tsx
import { useRole } from "@/hooks/use-role";

function RoleBadge() {
  const role = useRole();
  return <span>Peran: {role ?? "tamu"}</span>;
}
```

### `useHasRole`

Memeriksa apakah pengguna saat ini memiliki peran tertentu.

```tsx
import { useHasRole } from "@/hooks/use-has-role";

function AdminButton() {
  const isAdmin = useHasRole("admin");
  if (!isAdmin) return null;
  return <button>Aksi admin</button>;
}
```

| Param | Type | Deskripsi |
|-------|------|-----------|
| `role` | `string \| string[]` | Peran tunggal atau daftar peran yang diterima |

Mengembalikan `boolean`.
