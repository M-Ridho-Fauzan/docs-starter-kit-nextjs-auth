# Autentikasi Email & Password

Registrasi dan login email dan password melalui Better Auth.

## Konfigurasi

Atur di `auth.config.ts`:

```typescript
import { defineAuthConfig } from "@/auth/config";

export default defineAuthConfig({
  features: {
    emailPassword: {
      enabled: true,
      requireEmailVerification: false,
      passwordMinLength: 8,
      passwordMaxLength: 128,
      disableSignUp: false,
      autoSignIn: true,
    },
  },
});
```

| Opsi | Tipe | Standar | Deskripsi |
|------|------|---------|-----------|
| `enabled` | `boolean` | `false` | Aktifkan autentikasi email/password |
| `requireEmailVerification` | `boolean` | `false` | Wajibkan verifikasi email sebelum login |
| `passwordMinLength` | `number` | `8` | Panjang minimum password |
| `passwordMaxLength` | `number` | `128` | Panjang maksimum password |
| `disableSignUp` | `boolean` | `false` | Nonaktifkan registrasi pengguna baru |
| `autoSignIn` | `boolean` | `true` | Login otomatis setelah registrasi |

## Server Actions

### `signInWithEmailPassword`

Form action untuk login. Memvalidasi input, memanggil `auth.api.signInEmail()`, dan mengarahkan ulang jika berhasil.

```typescript
import { signInWithEmailPassword } from "@/lib/auth/actions/email-password";

<form action={signInWithEmailPassword}>
  <input name="email" type="email" required />
  <input name="password" type="password" required />
  <button type="submit">Sign in</button>
</form>
```

### `signUpWithEmailPassword`

Form action untuk registrasi. Memvalidasi input, memanggil `auth.api.signUpEmail()`, dan mengarahkan ulang jika berhasil.

```typescript
import { signUpWithEmailPassword } from "@/lib/auth/actions/email-password";

<form action={signUpWithEmailPassword}>
  <input name="name" type="text" required />
  <input name="email" type="email" required />
  <input name="password" type="password" required />
  <button type="submit">Create account</button>
</form>
```

## Hooks

### `useLoginForm`

```tsx
import { useLoginForm } from "@/hooks/use-login";

function LoginPage() {
  const [state, formAction, pending] = useLoginForm();
}
```

### `useRegisterForm`

```tsx
import { useRegisterForm } from "@/hooks/use-register";

function RegisterPage() {
  const [state, formAction, pending] = useRegisterForm();
}
```

## Tipe Return

```typescript
interface AuthSuccessResult {
  success: true;
  message?: string;
}

interface AuthErrorResult {
  success: false;
  error: {
    code: string;
    message: string;
  };
}

type AuthActionResult = AuthSuccessResult | AuthErrorResult;
```

## Kode Error

| Kode | Deskripsi |
|------|-----------|
| `VALIDATION_ERROR` | Field form tidak valid atau tidak diisi |
| `401` | Email atau password salah |
| `422` | Email sudah terdaftar (pendaftaran) |
| `429` | Terlalu banyak permintaan |
