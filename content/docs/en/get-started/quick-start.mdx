# Quick Start

From zero to running app in 5 minutes.

## 1. Create and Install

```bash
git clone https://github.com/your-org/starterkit-auth-nextjs.git my-app
cd my-app
pnpm install
```

## 2. Set Environment Variables

```bash
cp .env.example .env
```

Edit `.env` — set `DATABASE_URL` and `BETTER_AUTH_SECRET`.

## 3. Sync Database

```bash
npx prisma db push
```

## 4. Start

```bash
pnpm dev
```

Your app is running at [http://localhost:3000](http://localhost:3000).

## Default Features (out of the box)

- **Email/password** authentication — register at `/register`, sign in at `/login`
- **Password reset** — use `/forgot-password`
- **Dashboard** — protected route at `/dashboard`
- **JWT sessions** — stateless, no database reads on each request
- **Prisma adapter** — PostgreSQL database
- **shadcn/ui** — Tailwind CSS v4 components
- **Dark mode** — CSS variables ready with `.dark` class support

## Disabled by Default (enable in `auth.config.ts`)

| Feature | How to enable |
|---------|---------------|
| Email verification | Set `requireEmailVerification: true` |
| GitHub/Google login | Set OAuth provider `enabled: true` + credentials |
| Two-factor auth | Set `twoFactor.enabled: true` |
| Roles/permissions | Set `roles.enabled: true` |
| Database sessions | Set `session.strategy: "database"` |
