# Previous/Next Navigation Cards — 2026-06-25

**Goal**: Add card buttons at the bottom of each doc page showing Previous/Next page with title and description.

**Implementation**:
- Uses `findNeighbour` from `fumadocs-core/page-tree` — built-in utility
- `source.getPageTree()` → `findNeighbour(tree, page.url)` → `{ previous?: Item, next?: Item }`
- Navigation scoped per section via default `separateRoot: true`

**Files**:
- `components/docs/prev-next.tsx` — new component, renders two cards in flex row
- `app/[locale]/docs/[[...slug]]/page.tsx` — added `findNeighbour` call + `<PrevNext>`

**Card layout**: bordered card with arrow icon, "Previous"/"Next" label, page name (bold), description (muted, single line). Previous on left, Next on right.

**Build**: ✅ 85 pages, Tests: ✅ 166/166
